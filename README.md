# TIF & AI to PNG Premium Converter

브라우저 로컬에서 안전하고 빠르게 작동하는 고화질 이미지 변환기입니다. TIF/TIFF 이미지 파일 및 Adobe Illustrator(.ai) 벡터 그래픽스 파일을 PNG 형식으로 변환할 수 있습니다.

## 주요 기능

- **100% 로컬 보안 변환**: 파일이 서버로 업로드되지 않고 사용자의 PC 브라우저 내부 메모리에서 변환이 완료되므로 보안 유출 우려가 전혀 없습니다.
- **TIF 변환기 & AI 변환기 분리**: 상단 탭을 통해 간편하게 변환기를 전환하며 독립된 대기열과 설정을 가질 수 있습니다.
- **다중 대지(Artboard) 개별 추출**: AI 파일의 경우, 파일 내부의 모든 대지를 자동으로 감지하여 개별 PNG 파일로 각각 분할 추출합니다.
- **고해상도 벡터 렌더링**: AI 파일 변환 시 화질(72 DPI, 150 DPI, 300 DPI, 450 DPI)을 직접 선택하여 초고화질 이미지 출력이 가능합니다.
- **고급 가공 필터**:
  - 투명 배경 유지 / 흰색, 검은색 및 사용자 지정 색상 배경 채우기 지원
  - 흰색 배경 자동 제거(크로마키 오차 조정) 기능
  - 가로/세로 최대 크기 제한(Aspect Ratio 유지) 리사이징 기능
  - 파일명 커스텀 접미사(Suffix) 일괄 지정
- **일괄 다운로드**: 성공적으로 변환된 여러 개의 이미지들을 JSZip을 통해 원클릭 ZIP 파일 압축으로 한 번에 내려받을 수 있습니다.

## 사용법

1. `index.html` 파일을 크롬 등 브라우저로 엽니다.
2. 상단 탭에서 **TIF 변환기** 또는 **AI 변환기**를 선택합니다.
3. 왼쪽 사이드바에서 배경 색상, 화질(DPI), 리사이징 크기, 파일명 접미사 등의 원하는 옵션을 지정합니다.
4. 변환할 파일을 오른쪽 영역에 드래그 앤 드롭하거나 영역을 클릭하여 선택합니다.
5. **[일괄 변환 시작]** 버튼을 클릭하여 변환을 수행합니다.
6. 완료 후 개별 다운로드 버튼을 누르거나 **[선택 ZIP 다운로드]** 버튼을 눌러 압축 파일로 다운로드합니다.

## 사용 라이브러리 (CDN)

- [UTIF.js](https://github.com/photopea/UTIF.js) - TIF 이미지 구조 디코딩
- [PDF.js](https://mozilla.github.io/pdf.js/) - AI(PDF 호환) 벡터 그래픽스 파싱 및 캔버스 렌더링
- [JSZip](https://stuk.github.io/jszip/) - 변환 완료 이미지 일괄 압축 패키징
